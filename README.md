# Proyecto de Gestión de Proyectos - DUOC

Este repositorio contiene la documentación y entregables del proyecto de Gestión de Proyectos del DUOC.

## Contenido

El sitio web incluye:

- **Página Principal**: Vista general del proyecto
- **Carta Gantt**: Cronograma del proyecto
- **EDT (WBS)**: Estructura de Desglose del Trabajo
- **Matriz de Trazabilidad**: Seguimiento de requisitos
- **Alcance del Proyecto**: Definición del alcance
- **Requisitos**: Documentación de requisitos
- **Acta de Constitución**: Documento fundacional del proyecto

## Estructura del Proyecto

```
Semana 3/
├── web/                    # Sitio web del proyecto
│   ├── index.html         # Página principal
│   ├── gantt.html         # Carta Gantt
│   ├── edt.html           # EDT
│   ├── matriz_trazabilidad.html
│   ├── alcance.html
│   ├── requisitos.html
│   ├── acta_constitucion.html
│   ├── css/               # Estilos
│   └── js/                # Scripts
└── Documentos_descargables/  # Documentos Word/Excel originales
```

Este sitio está desplegado públicamente en Vercel.

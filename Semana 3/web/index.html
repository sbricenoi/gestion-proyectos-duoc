📌 Sobre este Proyecto

Caso de negocio: Konecta, empresa de telecomunicaciones con presencia en Chile, Argentina, Brasil y Colombia, necesita modernizar su intranet corporativa para mejorar la comunicación, centralizar información regionalizada y automatizar procesos de RRHH.

Objetivo: Desarrollar una plataforma web que sirva como canal único de comunicación y autoservicio para todos los empleados, logrando una adopción del 100%.

🚀 Resumen Ejecutivo del Proyecto

Contexto

Konecta es una empresa de telecomunicaciones con 2,700 empleados distribuidos en:

  • Chile: 1,500 empleados
  • Argentina: 300 empleados
  • Brasil: 500 empleados
  • Colombia: 400 empleados

Problemática

La intranet actual tiene baja adopción (44% de empleados fuera de Chile no la utilizan) debido a:

  • Contenido no regionalizado
  • Falta de funcionalidades transaccionales
  • Múltiples canales de comunicación dispersos
  • Procesos manuales que podrían ser automatizados

Solución Propuesta

Desarrollar una plataforma web moderna con:

  • Portal Regionalizado: Contenido específico por país
  • Autoservicio RRHH: Solicitud de vacaciones, liquidaciones, certificados
  • Comunicaciones: Canal único para anuncios, noticias y encuestas
  • Directorio Corporativo: Búsqueda de empleados y organigrama interactivo
  • Acceso Móvil: Diseño responsive para todos los dispositivos
  • Integración: Conexión con Active Directory, RRHH y Nómina

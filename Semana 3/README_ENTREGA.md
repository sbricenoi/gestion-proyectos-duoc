Los archivos están en formato Markdown (.md). Puedes:
- **Opción A**: Entregarlos en formato .md (son archivos de texto, se abren con cualquier editor)
- **Opción B**: Convertirlos a PDF usando:
  - Pandoc: `pandoc archivo.md -o archivo.pdf`
  - Markdown editors online: Dillinger.io, StackEdit, etc.
  - VS Code con extensión Markdown PDF

### Paso 4: Crear el ZIP
1. Crea una carpeta con nombre: `GPY2202_Exp1_S3_grupo15`
2. Incluye:
   - ✅ Acta de Constitución (PDF o MD)
